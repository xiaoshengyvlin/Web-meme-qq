# ✨ 表情包生成器

一个功能完整、界面精美的表情包生成器 Web 应用。

## 🎨 界面特色

- **清新唯美风格**：采用柔和的紫粉蓝渐变配色，视觉舒适
- **毛玻璃效果**：现代化的玻璃拟态设计
- **流畅动画**：浮动、发光、淡入等优雅的动画效果
- **响应式设计**：完美适配桌面端和移动端
- **优秀交互**：悬停效果、平滑过渡、即时反馈

## 🚀 功能特性

- 📦 **200+ 表情包模板**：内置丰富的模板库
- 🔍 **智能搜索**：支持按关键词搜索模板
- 🖼️ **多种图片输入**：本地上传、QQ头像、网络图片
- 📝 **灵活文字配置**：支持多段自定义文字
- 💾 **一键下载**：生成后直接下载
- 📋 **复制到剪贴板**：快速分享
- ⚡ **快速生成**：高性能后端处理

## 📦 安装与运行

### 环境要求
- Python 3.8+
- pip 包管理器

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动服务
```bash
python app.py
```

### 访问应用
打开浏览器访问：`http://localhost:7861`

## 🎯 使用说明

1. 在左侧选择一个表情包模板
2. 根据模板要求上传图片（可选）
3. 输入想要的文字内容
4. 点击"生成表情包"按钮
5. 生成完成后可下载或复制
